# js-html-to-markdown

A function for convert html string to markdown.

## Overview
The function is used for convert a html input to a markdown output.

## Installation
No installation required! Simply include the function in your JavaScript project.

```javascript
import { htmlToMarkdown } from 'js-html-to-markdown';
```

## Usage
Converting innerHTML to markdown:
```javascript
const div = document.getElementById('my-div');
const markdown = htmlToMarkdown(div.innerHTML);
```
